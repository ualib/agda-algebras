# Provenance — Interval enforceable properties of finite groups

This directory vendors the source of the research note *Interval enforceable properties of finite groups* by William J. DeMeo: the LaTeX source `IEProps-1205.1927v4.tex`, the generated bibliography `IEProps-1205.1927v4.bbl`, and the compiled `IEProps-1205.1927v4.pdf`.  It corresponds to version 4 of arXiv:1205.1927, https://arxiv.org/abs/1205.1927.

+  **Why it is vendored**.  The note is the mathematical basis of the FLRP research program tracked in `docs/notes/flrp-research-roadmap.md` and issues #451–#461; the formalization work packages cite its definitions and numbered results directly, so the exact source text is kept in-repo as the stable reference.
