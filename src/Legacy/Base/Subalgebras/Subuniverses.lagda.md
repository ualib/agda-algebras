### <a id="subuniverses">Subuniverses</a>

This is the [Base.Subalgebras.Subuniverses][] module of the [Agda Universal Algebra Library][].

We start by defining a type that represents the important concept of *subuniverse*. Suppose `рқ‘Ё` is an algebra.  A subset `B вҠҶ вҲЈ рқ‘Ё вҲЈ` is said to be *closed under the operations of* `рқ‘Ё` if for each `рқ‘“ вҲҲ вҲЈ рқ‘Ҷ вҲЈ` and all tuples `рқ’ғ : вҲҘ рқ‘Ҷ вҲҘ рқ‘“ вҶ’ рқҗө` the element `(рқ‘“ МӮ рқ‘Ё) рқ’ғ` belongs to `B`. If a subset `B вҠҶ рқҗҙ` is closed under the operations of `рқ‘Ё`, then we call B a *subuniverse* of `рқ‘Ё`.


```agda


{-# OPTIONS --cubical-compatible --exact-split --safe #-}

open import Overture using ( рқ“һ ; рқ“Ҙ ; Signature )

module Legacy.Base.Subalgebras.Subuniverses {рқ‘Ҷ : Signature рқ“һ рқ“Ҙ} where

-- Imports from Agda and the Agda Standard Library -----------------------------

open import Function             using ( _вҲҳ_ )
open import Level                using ( Level ; _вҠ”_ )

open import Axiom.Extensionality.Propositional
                                 using () renaming ( Extensionality to funext )
open import Relation.Binary.PropositionalEquality
                                 using ( module вүЎ-Reasoning ; _вүЎ_ )

-- Imports from the Agda Universal Algebra Library -----------------------------
open import Overture                     using ( вҲЈ_вҲЈ ; вҲҘ_вҲҘ ; _вҒ»В№ )
open import Legacy.Base.Relations               using ( Im_вҠҶ_ )
open import Legacy.Base.Equality                using ( swelldef )
open import Legacy.Base.Algebras       {рқ‘Ҷ = рқ‘Ҷ}  using ( Algebra ; _МӮ_ ; ov )
open import Legacy.Base.Homomorphisms  {рқ‘Ҷ = рқ‘Ҷ}  using ( hom )
open import Legacy.Base.Terms          {рқ‘Ҷ = рқ‘Ҷ}  using ( Term ; в„Ҡ ; node ; _вҹҰ_вҹ§ )

```


We first show how to represent in [Agda][] the collection of subuniverses of an
algebra `рқ‘Ё`. Since a subuniverse is viewed as a subset of the domain of `рқ‘Ё`,
we define it as a predicate on `вҲЈ рқ‘Ё вҲЈ`.  Thus, the collection of subuniverses
is a predicate on predicates on `вҲЈ рқ‘Ё вҲЈ`.


```agda


Subuniverses : (рқ‘Ё : Algebra Оұ) вҶ’ Pred (Pred вҲЈ рқ‘Ё вҲЈ ОІ) (рқ“һ вҠ” рқ“Ҙ вҠ” Оұ вҠ” ОІ)
Subuniverses рқ‘Ё B = (рқ‘“ : вҲЈ рқ‘Ҷ вҲЈ)(рқ‘Һ : вҲҘ рқ‘Ҷ вҲҘ рқ‘“ вҶ’ вҲЈ рқ‘Ё вҲЈ) вҶ’ Im рқ‘Һ вҠҶ B вҶ’ (рқ‘“ МӮ рқ‘Ё) рқ‘Һ вҲҲ B
```



#### <a id="subuniverses-as-records">Subuniverses as records</a>

Next we define a type to represent a single subuniverse of an algebra. If `рқ‘Ё`
is the algebra in question, then a subuniverse of `рқ‘Ё` is a subset of (i.e.,
predicate over) the domain `вҲЈ рқ‘Ё вҲЈ` that belongs to `Subuniverses рқ‘Ё`.


```agda


record Subuniverse {рқ‘Ё : Algebra Оұ} : Type(ov ОІ вҠ” Оұ) where
 constructor mksub
 field
  sset  : Pred вҲЈ рқ‘Ё вҲЈ ОІ
  sSub : sset вҲҲ Subuniverses рқ‘Ё
```



#### <a id="subuniverse-generation">Subuniverse Generation</a>

If `рқ‘Ё` is an algebra and `X вҠҶ вҲЈ рқ‘Ё вҲЈ` a subset of the domain of `рқ‘Ё`, then the
*subuniverse of* `рқ‘Ё` *generated by* `X` is typically denoted by
`Sg`<sup>`рқ‘Ё`</sup>`(X)` and defined to be the smallest subuniverse of `рқ‘Ё`
containing `X`.  Equivalently,

`Sg`<sup>`рқ‘Ё`</sup>`(X)`  =  `вӢӮ` { `U` : `U` is a subuniverse of `рқ‘Ё` and  `B вҠҶ U` }.

We define an inductive type, denoted by `Sg`, that represents the subuniverse
generated by a given subset of the domain of a given algebra, as follows.


```agda


data Sg (рқ‘Ё : Algebra Оұ)(X : Pred вҲЈ рқ‘Ё вҲЈ ОІ) : Pred вҲЈ рқ‘Ё вҲЈ (рқ“һ вҠ” рқ“Ҙ вҠ” Оұ вҠ” ОІ)
 where
 var : вҲҖ {v} вҶ’ v вҲҲ X вҶ’ v вҲҲ Sg рқ‘Ё X
 app : вҲҖ f a вҶ’ Im a вҠҶ Sg рқ‘Ё X вҶ’ (f МӮ рқ‘Ё) a вҲҲ Sg рқ‘Ё X
```


(The inferred types in the `app` constructor are `f : вҲЈ рқ‘Ҷ вҲЈ` and `a : вҲҘ рқ‘Ҷ вҲҘ рқ‘“ вҶ’ вҲЈ рқ‘Ё вҲЈ`.)

Given an arbitrary subset `X` of the domain `вҲЈ рқ‘Ё вҲЈ` of an `рқ‘Ҷ`-algebra `рқ‘Ё`, the type
`Sg X` does indeed represent a subuniverse of `рқ‘Ё`. Proving this using the inductive
type `Sg` is trivial, as we see here.


```agda


sgIsSub : {рқ‘Ё : Algebra Оұ}{X : Pred вҲЈ рқ‘Ё вҲЈ ОІ} вҶ’ Sg рқ‘Ё X вҲҲ Subuniverses рқ‘Ё
sgIsSub = app
```


Next we prove by structural induction that `Sg X` is the smallest subuniverse
of `рқ‘Ё` containing `X`.


```agda


sgIsSmallest :  {рқ“Ў : Level}(рқ‘Ё : Algebra Оұ){X : Pred вҲЈ рқ‘Ё вҲЈ ОІ}(Y : Pred вҲЈ рқ‘Ё вҲЈ рқ“Ў)
 вҶ’              Y вҲҲ Subuniverses рқ‘Ё  вҶ’  X вҠҶ Y  вҶ’  Sg рқ‘Ё X вҠҶ Y

sgIsSmallest _ _ _ XinY (var Xv) = XinY Xv
sgIsSmallest рқ‘Ё Y YsubA XinY (app f a SgXa) = Yfa
 where
 IH : Im a вҠҶ Y
 IH i = sgIsSmallest рқ‘Ё Y YsubA XinY (SgXa i)

 Yfa : (f МӮ рқ‘Ё) a вҲҲ Y
 Yfa = YsubA f a IH
```


When the element of `Sg X` is constructed as `app f a SgXa`, we may assume (the
induction hypothesis) that the arguments in the tuple `a` belong to `Y`. Then
the result of applying `f` to `a` also belongs to `Y` since `Y` is a subuniverse.


#### <a id="subuniverse-lemmas">Subuniverse Lemmas</a>

Here we formalize a few basic properties of subuniverses. First, the intersection
of subuniverses is again a subuniverse.


```agda


вӢӮs :  {рқ“ҳ : Level}{рқ‘Ё : Algebra Оұ}{I : Type рқ“ҳ}{рқ’ң : I вҶ’ Pred вҲЈ рқ‘Ё вҲЈ ОІ}
 вҶ’    (вҲҖ i вҶ’ рқ’ң i вҲҲ Subuniverses рқ‘Ё) вҶ’ вӢӮ I рқ’ң вҲҲ Subuniverses рқ‘Ё

вӢӮs Пғ f a ОҪ = О» i вҶ’ Пғ i f a (О» x вҶ’ ОҪ x i)
```


In the proof above, we assume the following typing judgments:


    Пғ : вҲҖ i вҶ’ рқ’ң i вҲҲ Subuniverses рқ‘Ё
    f : вҲЈ рқ‘Ҷ вҲЈ
    a : вҲҘ рқ‘Ҷ вҲҘ рқ‘“ вҶ’ вҲЈ рқ‘Ё вҲЈ
    ОҪ : Im рқ‘Һ вҠҶ вӢӮ I рқ’ң

and we must prove `(f МӮ рқ‘Ё) a вҲҲ вӢӮ I рқ’ң`. In this case, Agda will fill in the proof
term `О» i вҶ’ Пғ i f a (О» x вҶ’ ОҪ x i)` automatically with the command `C-c C-a`.

Next, subuniverses are closed under the action of term operations.


```agda


sub-term-closed :  {рқ“§ : Level}{X : Type рқ“§}(рқ‘Ё : Algebra Оұ){B : Pred вҲЈ рқ‘Ё вҲЈ ОІ}
 вҶ’                 (B вҲҲ Subuniverses рқ‘Ё) вҶ’ (t : Term X)(b : X вҶ’ вҲЈ рқ‘Ё вҲЈ)
 вҶ’                 ((x : X) вҶ’ (b x вҲҲ B)) вҶ’ (рқ‘Ё вҹҰ t вҹ§)b вҲҲ B

sub-term-closed рқ‘Ё AB (в„Ҡ x) b Bb = Bb x

sub-term-closed рқ‘Ё{B} Пғ (node f t)b ОҪ =
 Пғ f  (О» z вҶ’ (рқ‘Ё вҹҰ t z вҹ§) b) О» x вҶ’ sub-term-closed рқ‘Ё{B} Пғ (t x) b ОҪ
```


In the induction step of the foregoing proof, the typing judgments of the premise
are the following:

    рқ‘Ё   : Algebra Оұ
    B   : Pred вҲЈ рқ‘Ё вҲЈ ОІ
    Пғ   : B вҲҲ Subuniverses рқ‘Ё
    f   : вҲЈ рқ‘Ҷ вҲЈ
    t   : вҲҘ рқ‘Ҷ вҲҘ рқ‘“ вҶ’ Term X
    b   : X вҶ’ вҲЈ рқ‘Ё вҲЈ
    ОҪ   : вҲҖ x вҶ’ b x вҲҲ B

and the given proof term establishes the goal `рқ‘Ё вҹҰ node f t вҹ§ b вҲҲ B`.

Alternatively, we could express the preceeding fact using an inductive type
representing images of terms.


```agda


data TermImage (рқ‘Ё : Algebra Оұ)(Y : Pred вҲЈ рқ‘Ё вҲЈ ОІ) : Pred вҲЈ рқ‘Ё вҲЈ (рқ“һ вҠ” рқ“Ҙ вҠ” Оұ вҠ” ОІ)
 where
 var : вҲҖ {y : вҲЈ рқ‘Ё вҲЈ} вҶ’ y вҲҲ Y вҶ’ y вҲҲ TermImage рқ‘Ё Y
 app : вҲҖ рқ‘“ рқ‘Ў вҶ’  ((x : вҲҘ рқ‘Ҷ вҲҘ рқ‘“) вҶ’ рқ‘Ў x вҲҲ TermImage рқ‘Ё Y)  вҶ’ (рқ‘“ МӮ рқ‘Ё) рқ‘Ў вҲҲ TermImage рқ‘Ё Y
```


By what we proved above, it should come as no surprise that `TermImage рқ‘Ё Y` is a
subuniverse of `рқ‘Ё` that contains `Y`.


```agda


TermImageIsSub : {рқ‘Ё : Algebra Оұ}{Y : Pred вҲЈ рқ‘Ё вҲЈ ОІ} вҶ’ TermImage рқ‘Ё Y вҲҲ Subuniverses рқ‘Ё
TermImageIsSub = app

Y-onlyif-TermImageY : {рқ‘Ё : Algebra Оұ}{Y : Pred вҲЈ рқ‘Ё вҲЈ ОІ} вҶ’ Y вҠҶ TermImage рқ‘Ё Y
Y-onlyif-TermImageY {a} Ya = var Ya
```


Since `Sg рқ‘Ё Y` is the smallest subuniverse containing Y, we obtain the following inclusion.


```agda


SgY-onlyif-TermImageY : (рқ‘Ё : Algebra Оұ)(Y : Pred вҲЈ рқ‘Ё вҲЈ ОІ) вҶ’ Sg рқ‘Ё Y вҠҶ TermImage рқ‘Ё Y
SgY-onlyif-TermImageY рқ‘Ё Y = sgIsSmallest рқ‘Ё (TermImage рқ‘Ё Y) TermImageIsSub Y-onlyif-TermImageY
```


Next we prove the important fact that homomorphisms are uniquely determined by their
values on a generating set.


```agda


open вүЎ-Reasoning

hom-unique :  swelldef рқ“Ҙ ОІ вҶ’ {рқ‘Ё : Algebra Оұ}{рқ‘© : Algebra ОІ}
              (X : Pred вҲЈ рқ‘Ё вҲЈ Оұ)  (g h : hom рқ‘Ё рқ‘©)
 вҶ’            ((x : вҲЈ рқ‘Ё вҲЈ) вҶ’ (x вҲҲ X вҶ’ вҲЈ g вҲЈ x вүЎ вҲЈ h вҲЈ x))
              -----------------------------------------------
 вҶ’            (a : вҲЈ рқ‘Ё вҲЈ) вҶ’ (a вҲҲ Sg рқ‘Ё X вҶ’ вҲЈ g вҲЈ a вүЎ вҲЈ h вҲЈ a)

hom-unique _ _ _ _ Пғ a (var x) = Пғ a x

hom-unique wd {рқ‘Ё}{рқ‘©} X g h Пғ fa (app рқ‘“ a ОҪ) = Goal
 where
 IH : вҲҖ x вҶ’ вҲЈ g вҲЈ (a x) вүЎ вҲЈ h вҲЈ (a x)
 IH x = hom-unique wd{рқ‘Ё}{рқ‘©} X g h Пғ (a x) (ОҪ x)

 Goal : вҲЈ g вҲЈ ((рқ‘“ МӮ рқ‘Ё) a) вүЎ вҲЈ h вҲЈ ((рқ‘“ МӮ рқ‘Ё) a)
 Goal =  вҲЈ g вҲЈ ((рқ‘“ МӮ рқ‘Ё) a)    вүЎвҹЁ вҲҘ g вҲҘ рқ‘“ a вҹ©
         (рқ‘“ МӮ рқ‘©)(вҲЈ g вҲЈ вҲҳ a )  вүЎвҹЁ wd (рқ‘“ МӮ рқ‘©) (вҲЈ g вҲЈ вҲҳ a) (вҲЈ h вҲЈ вҲҳ a) IH вҹ©
         (рқ‘“ МӮ рқ‘©)(вҲЈ h вҲЈ вҲҳ a)   вүЎвҹЁ ( вҲҘ h вҲҘ рқ‘“ a )вҒ»В№ вҹ©
         вҲЈ h вҲЈ ((рқ‘“ МӮ рқ‘Ё) a )   вҲҺ
```


In the induction step, the following typing judgments are assumed:

    wd  : swelldef рқ“Ҙ ОІ
    рқ‘Ё   : Algebra Оұ
    рқ‘©   : Algebra ОІ
    X   : Pred вҲЈ рқ‘Ё вҲЈ Оұ
    g h  : hom рқ‘Ё рқ‘©
    Пғ   : О  x кһү вҲЈ рқ‘Ё вҲЈ , (x вҲҲ X вҶ’ вҲЈ g вҲЈ x вүЎ вҲЈ h вҲЈ x)
    fa  : вҲЈ рқ‘Ё вҲЈ
    fa  = (рқ‘“ МӮ рқ‘Ё) a
    рқ‘“   : вҲЈ рқ‘Ҷ вҲЈ
    a   : вҲҘ рқ‘Ҷ вҲҘ рқ‘“ вҶ’ вҲЈ рқ‘Ё вҲЈ
    ОҪ   : Im a вҠҶ Sg рқ‘Ё X

and, under these assumptions, we proved `вҲЈ g вҲЈ ((рқ‘“ МӮ рқ‘Ё) a) вүЎ вҲЈ h вҲЈ ((рқ‘“ МӮ рқ‘Ё) a)`.
