### The Complete Lattice of Subuniverses

This is the [Setoid.Subalgebras.CompleteLattice][] module of the [Agda Universal Algebra Library][].

The subuniverses of a setoid algebra `ūĚĎ®` ‚ÄĒ the subsets of its carrier closed under
the basic operations ‚ÄĒ form a **complete lattice** `Sub ūĚĎ®` under inclusion.  This is
the second instance of the order-theoretic `CompleteLattice` record of
[Order.CompleteLattice][] (the first being the congruence lattice
[Setoid.Congruences.CompleteLattice][]), and it is built directly from the
subuniverse-generation machinery of [Setoid.Subalgebras.Subuniverses][]:

+  `Subuniverses ūĚĎ®` ‚ÄĒ the predicate "is a subuniverse";
+  `Sg ūĚĎ® G` ‚ÄĒ the subuniverse generated by `G`, with `sgIsSub` (it is a subuniverse)
   and `sgIsSmallest` (it is the *least* subuniverse containing `G`);
+  `‚čās` ‚ÄĒ an arbitrary intersection of subuniverses is a subuniverse.

As with the congruence lattice the join generated by `Sg` raises the predicate level
from `‚Ąď` to `ūĚďě ‚äĒ ūĚď• ‚äĒ őĪ ‚äĒ ‚Ąď`, so we evaluate the lattice at the absorbing level
`L = ūĚďě ‚äĒ ūĚď• ‚äĒ őĪ ‚äĒ ‚Ąď‚āÄ`, at which `Sg` (and the infinitary `‚čÉ`/`‚čā` over `‚Ąď‚āÄ`-small
families) stays at `L`.  (Unlike congruences there is no `ŌĀ`, since a subuniverse is
a predicate on the carrier and does not mention the setoid equality.)

#### The subuniverse lattice at the absorbing level `L`

The subuniverse lattice of an algebra is formalized here and packaged inside a module
called `Sublattice`, which is parametrized by the algebra `ūĚĎ®` and a base level `‚Ąď‚āÄ`.
This way opening the `Sublattice` module at a use site (with, e.g., `open Sublattice ūĚĎ® ‚Ąď‚āÄ`)
makes available `_‚Č§_`, `_‚ąß_`, `_‚ą®_`, the bounds, and the bundles specialized to `ūĚĎ®`.
One can then write `B ‚Č§ C`, instead of `_‚Č§_ ūĚĎ® ‚Ąď‚āÄ B C`.

```agda
module Sublattice {ūĚďě ūĚď• : Level}{ūĚĎÜ : Signature ūĚďě ūĚď•}(ūĚĎ® : Algebra {ūĚĎÜ = ūĚĎÜ} őĪ ŌĀŠĶÉ) (‚Ąď‚āÄ : Level) where
  private A = ūĚēĆ[ ūĚĎ® ]

  L : Level
  L = ūĚďě ‚äĒ ūĚď• ‚äĒ őĪ ‚äĒ ‚Ąď‚āÄ

  -- A subuniverse, as its underlying predicate together with a proof of closure.
  SubŠīł : Type (őĪ ‚äĒ ov {ūĚĎÜ = ūĚĎÜ} L)
  SubŠīł = ő£[ B ‚ąą Pred A L ] B ‚ąą Subuniverses ūĚĎ®
```

The order is inclusion of the underlying predicates, and the associated equality is
mutual inclusion.

```agda
  infix 4 _‚Č§_ _‚Čą_

  _‚Č§_ : SubŠīł ‚Üí SubŠīł ‚Üí Type (őĪ ‚äĒ L)
  B ‚Č§ C = proj‚āĀ B ‚äÜ proj‚āĀ C

  _‚Čą_ : SubŠīł ‚Üí SubŠīł ‚Üí Type (őĪ ‚äĒ L)
  B ‚Čą C = (B ‚Č§ C) √ó (C ‚Č§ B)

  -- The proofs are inlined (rather than routed through named ‚Č§-lemmas) because
  -- _‚Č§_ is a defined relation whose congruence arguments Agda cannot recover.
  ‚Čą-isEquivalence : IsEquivalence _‚Čą_
  ‚Čą-isEquivalence = record
   { refl   = (őĽ z ‚Üí z) , (őĽ z ‚Üí z)
   ; sym    = őĽ (p , q) ‚Üí q , p
   ; trans  = őĽ (p , q) (p‚Ä≤ , q‚Ä≤) ‚Üí (őĽ z ‚Üí p‚Ä≤ (p z)) , (őĽ z ‚Üí q (q‚Ä≤ z))
   }

  ‚Č§-isPartialOrder : IsPartialOrder _‚Čą_ _‚Č§_
  ‚Č§-isPartialOrder = record
   { isPreorder = record  { isEquivalence  = ‚Čą-isEquivalence
                          ; reflexive      = proj‚āĀ
                          ; trans          = őĽ p q z ‚Üí q (p z)
                          }
   ; antisym = őĽ p q ‚Üí p , q
   }
```

#### Meet and join

The meet is the intersection of the underlying predicates (a subuniverse,
componentwise), and the join is the subuniverse *generated* by the union.

```agda
  infixr 7 _‚ąß_
  infixr 6 _‚ą®_

  _‚ąß_ : SubŠīł ‚Üí SubŠīł ‚Üí SubŠīł
  B ‚ąß C = (proj‚āĀ B ‚ą© proj‚āĀ C)
        , őĽ f a im ‚Üí proj‚āā B f a (őĽ i ‚Üí proj‚āĀ (im i))
                   , proj‚āā C f a (őĽ i ‚Üí proj‚āā (im i))

  _‚ą®_ : SubŠīł ‚Üí SubŠīł ‚Üí SubŠīł
  B ‚ą® C = Sg ūĚĎ® (proj‚āĀ B ‚ą™ proj‚āĀ C) , sgIsSub ūĚĎ®

  -- The meet is the greatest lower bound.
  ‚ąß-infimum : Infimum _‚Č§_ _‚ąß_
  ‚ąß-infimum B C =  (őĽ z ‚Üí proj‚āĀ z)
                ,  (őĽ z ‚Üí proj‚āā z)
                ,  őĽ D D‚Č§B D‚Č§C z ‚Üí D‚Č§B z , D‚Č§C z

  -- The join is the least upper bound (upper bounds via `var`, universality via
  -- `sgIsSmallest`, since the union is below any subuniverse above both arguments).
  ‚ą®-supremum : Supremum _‚Č§_ _‚ą®_
  ‚ą®-supremum B C =  (őĽ z ‚Üí var (inj‚āĀ z))
                 ,  (őĽ z ‚Üí var (inj‚āā z))
                 ,  őĽ D B‚Č§D C‚Č§D ‚Üí sgIsSmallest ūĚĎ® (proj‚āĀ D) (proj‚āā D)
                                    (őĽ { (inj‚āĀ x) ‚Üí B‚Č§D x ; (inj‚āā x) ‚Üí C‚Č§D x })
```

#### The bounds: empty and full subuniverses

The bottom subuniverse `0ňĘ` is the *least* subuniverse, obtained as the one generated
by the empty predicate (`0ňĘ = Sg ‚ąÖ`); its minimality is immediate from `sgIsSmallest`.
The top subuniverse `1ňĘ` is the whole carrier, trivially closed under the operations.

```agda
  private
    0R : Pred A L
    0R _ = Lift L ‚ä•

    1R : Pred A L
    1R _ = Lift L ‚ä§

  0ňĘ : SubŠīł

  1ňĘ : SubŠīł
  1ňĘ = 1R , őĽ _ _ _ ‚Üí lift tt

  0ňĘ-minimum : Minimum _‚Č§_ 0ňĘ
  0ňĘ-minimum B = sgIsSmallest ūĚĎ® (proj‚āĀ B) (proj‚āā B) (őĽ { (lift ()) })

  1ňĘ-maximum : Maximum _‚Č§_ 1ňĘ
  1ňĘ-maximum B _ = lift tt

  Sub-isBoundedLattice : IsBoundedLattice _‚Čą_ _‚Č§_ _‚ą®_ _‚ąß_ 1ňĘ 0ňĘ
  Sub-isBoundedLattice = record  { isLattice  = Sub-isLattice
                                 ; maximum    = 1ňĘ-maximum
                                 ; minimum    = 0ňĘ-minimum
                                 }

  Sub-BoundedLattice : BoundedLattice (őĪ ‚äĒ ov {ūĚĎÜ = ūĚĎÜ} L) (őĪ ‚äĒ L) (őĪ ‚äĒ L)
  Sub-BoundedLattice = record  { Carrier           = SubŠīł
                               ; _‚Čą_               = _‚Čą_

                               ; _‚ą®_               = _‚ą®_
                               ; _‚ąß_               = _‚ąß_

                               ; ‚ä•                 = 0ňĘ

```

#### Infinitary meets and joins

For a family `ūĚíú : I ‚Üí SubŠīł` indexed by `I : Type ‚Ąď‚āÄ`, the infinitary meet is the
intersection `‚®Ö ūĚíú` (which holds at `x` iff every `ūĚíú i` does), and the infinitary join
is the subuniverse generated by the union, `‚®Ü ūĚíú = Sg(‚čÉ ūĚíú)`.  Both stay at level `L`
because `I` is `‚Ąď‚āÄ`-small.

```agda
  ‚®Ö : {I : Type ‚Ąď‚āÄ} ‚Üí (I ‚Üí SubŠīł) ‚Üí SubŠīł
  ‚®Ö {I} ūĚíú = ‚čā I (őĽ i ‚Üí proj‚āĀ (ūĚíú i))
           , ‚čās {ūĚĎ® = ūĚĎ®} I {ūĚíú = őĽ i ‚Üí proj‚āĀ (ūĚíú i)} (őĽ i ‚Üí proj‚āā (ūĚíú i))

  ‚®Ü : {I : Type ‚Ąď‚āÄ} ‚Üí (I ‚Üí SubŠīł) ‚Üí SubŠīł
  ‚®Ü {I} ūĚíú = Sg ūĚĎ® (‚čÉ I (őĽ i ‚Üí proj‚āĀ (ūĚíú i))) , sgIsSub ūĚĎ®

  ‚®Ö-lower : {I : Type ‚Ąď‚āÄ} (ūĚíú : I ‚Üí SubŠīł) (i : I) ‚Üí (‚®Ö ūĚíú) ‚Č§ (ūĚíú i)
  ‚®Ö-lower ūĚíú i z = z i

  ‚®Ö-greatest : {I : Type ‚Ąď‚āÄ} (ūĚíú : I ‚Üí SubŠīł) (D : SubŠīł) ‚Üí (‚ąÄ i ‚Üí D ‚Č§ (ūĚíú i)) ‚Üí D ‚Č§ (‚®Ö ūĚíú)
  ‚®Ö-greatest ūĚíú D D‚Č§ūĚíú z i = D‚Č§ūĚíú i z

  ‚®Ü-upper : {I : Type ‚Ąď‚āÄ} (ūĚíú : I ‚Üí SubŠīł) (i : I) ‚Üí (ūĚíú i) ‚Č§ (‚®Ü ūĚíú)
  ‚®Ü-upper ūĚíú i z = var (i , z)

  ‚®Ü-least : {I : Type ‚Ąď‚āÄ} (ūĚíú : I ‚Üí SubŠīł) (D : SubŠīł) ‚Üí (‚ąÄ i ‚Üí (ūĚíú i) ‚Č§ D) ‚Üí (‚®Ü ūĚíú) ‚Č§ D
  ‚®Ü-least ūĚíú D ūĚíú‚Č§D = sgIsSmallest ūĚĎ® (proj‚āĀ D) (proj‚āā D) (őĽ (i , z) ‚Üí ūĚíú‚Č§D i z)
```
