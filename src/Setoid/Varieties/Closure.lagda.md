#### Closure Operators for Setoid Algebras

Fix a signature рқ‘Ҷ, let рқ’Ұ be a class of рқ‘Ҷ-algebras, and define

* H рқ’Ұ = algebras isomorphic to a homomorphic image of a members of рқ’Ұ;
* S рқ’Ұ = algebras isomorphic to a subalgebra of a member of рқ’Ұ;
* P рқ’Ұ = algebras isomorphic to a product of members of рқ’Ұ.


```agda
{-# OPTIONS --cubical-compatible --exact-split --safe #-}

open import Overture using (рқ“һ ; рқ“Ҙ ; Signature)

 H : вҲҖ в„“ вҶ’ Pred(Algebra Оұ ПҒбөғ) (a вҠ” ov в„“) вҶ’ Pred(Algebra ОІ ПҒбөҮ) (b вҠ” ov(a вҠ” в„“))

 SP в„“ О№ рқ’Ұ = S{Оұ}{ПҒбөғ} (a вҠ” в„“ вҠ” О№) (P в„“ О№ рқ’Ұ)

 V в„“ О№ рқ’Ұ = H{Оі}{ПҒб¶ң}{Оҙ}{ПҒбөҲ} (a вҠ” b вҠ” в„“ вҠ” О№) (S{ОІ}{ПҒбөҮ} (a вҠ” в„“ вҠ” О№) (P в„“ О№ рқ’Ұ))
```


Thus, if рқ’Ұ is a class of рқ‘Ҷ-algebras, then the *variety generated by* рқ’Ұ is denoted by `V рқ’Ұ` and defined to be the smallest class that contains рқ’Ұ and is closed under `H`, `S`, and `P`.

With the closure operator V representing closure under HSP, we represent formally what it means to be a variety of algebras as follows.


```agda
module _ {Оұ ПҒбөғ в„“ О№ : Level} where

 is-variety : (рқ’ұ : Pred (Algebra Оұ ПҒбөғ)(Оұ вҠ” ПҒбөғ вҠ” ov в„“)) вҶ’ Type (ov (Оұ вҠ” ПҒбөғ вҠ” в„“ вҠ” О№))
 is-variety рқ’ұ = V{ОІ = Оұ}{ПҒбөҮ = ПҒбөғ}{Оі = Оұ}{ПҒб¶ң = ПҒбөғ} в„“ О№ рқ’ұ вҠҶ рқ’ұ

 variety : Type (ov (Оұ вҠ” ПҒбөғ вҠ” ov в„“ вҠ” О№))
 variety = ОЈ[ рқ’ұ вҲҲ (Pred (Algebra Оұ ПҒбөғ) (Оұ вҠ” ПҒбөғ вҠ” ov в„“)) ] is-variety рқ’ұ
```


#### A common-case level specialization of `V`

The operator `V`{.AgdaFunction} carries *eight* independent universe levels.  The
input class lives at `(Оұ, ПҒбөғ)` and the output algebra at `(Оҙ, ПҒбөҲ)`, but the
intermediate pairs `(ОІ, ПҒбөҮ)` and `(Оі, ПҒб¶ң)` are the levels of the algebras threaded
through the `H вҲҳ S вҲҳ P` composition: they are determined by neither the input nor the
output, so they stay free metavariables whenever `V`{.AgdaFunction} is applied without
a goal that already pins them.  That generality is essential for the HSP theorem, but
it is a papercut for the everyday request "the variety generated by a fixed algebra".

`VвҖІ`{.AgdaFunction} is the specialization of `V`{.AgdaFunction} to the overwhelmingly
common case in which the generated variety is considered at the *same* levels as the
class that generates it.  It collapses all four algebra positions to the input levels
`(Оұ, ПҒбөғ)` вҖ” exactly the pinning `is-variety`{.AgdaFunction} already performs above
(`ОІ = Оі = Оҙ = Оұ` and `ПҒбөҮ = ПҒб¶ң = ПҒбөҲ = ПҒбөғ`), under which the output level
`d вҠ” ov(a вҠ” b вҠ” c вҠ” в„“ вҠ” О№)` of `V`{.AgdaFunction} collapses to `a вҠ” ov(a вҠ” в„“ вҠ” О№)`.
This is a documented narrowing of the canonical operator, not a synonym:
`V`{.AgdaFunction} remains the level-polymorphic form, while `VвҖІ`{.AgdaFunction} is the
fixed-level entry point that downstream examples and tests should prefer.

```agda
module _  {Оұ ПҒбөғ : Level} where
 private a = Оұ вҠ” ПҒбөғ

 VвҖІ : вҲҖ в„“ О№ вҶ’ Pred(Algebra Оұ ПҒбөғ)(a вҠ” ov в„“) вҶ’ Pred(Algebra Оұ ПҒбөғ) (a вҠ” ov(a вҠ” в„“ вҠ” О№))
 VвҖІ в„“ О№ рқ’Ұ = V {Оұ}{ПҒбөғ}{Оұ}{ПҒбөғ}{Оұ}{ПҒбөғ}{Оұ}{ПҒбөғ} в„“ О№ рқ’Ұ
```


#### Closure properties of S

`S` is a closure operator.  The fact that S is expansive won't be needed, so we omit the proof, but we will make use of monotonicity and idempotence of `S`.


```agda
module _ {Оұ ПҒбөғ : Level} where

 private a = Оұ вҠ” ПҒбөғ

 S-mono :  вҲҖ{в„“} вҶ’ {рқ’Ұ рқ’Ұ' : Pred (Algebra Оұ ПҒбөғ)(a вҠ” ov в„“)}
  вҶ’        рқ’Ұ вҠҶ рқ’Ұ' вҶ’ S{ОІ = Оұ}{ПҒбөғ} в„“ рқ’Ұ вҠҶ S в„“ рқ’Ұ'

 S-mono kk {рқ‘©} (рқ‘Ё , (kA , BвүӨA)) = рқ‘Ё , ((kk kA) , BвүӨA)
```


We say `S` is *idempotent* provided `S`{.AgdaFunction} (`S`{.AgdaFunction} `рқ’Ұ`{.AgdaBound}) `=`{.AgdaSymbol} `S`{.AgdaFunction} `рқ’Ұ`{.AgdaBound}.
Of course, this is proved by establishing two inclusions, but one of them is trivial, so only the other need be formalized, which we do as follows.


```agda
 S-idem :  вҲҖ{ОІ ПҒбөҮ Оі ПҒб¶ң в„“} вҶ’ {рқ’Ұ : Pred (Algebra Оұ ПҒбөғ)(a вҠ” ov в„“)}
  вҶ’        S{ОІ = Оі}{ПҒб¶ң} (a вҠ” в„“) (S{ОІ = ОІ}{ПҒбөҮ} в„“ рқ’Ұ) вҠҶ S{ОІ = Оі}{ПҒб¶ң} в„“ рқ’Ұ

 S-idem (рқ‘Ё , (рқ‘© , sB , AвүӨB) , xвүӨA) = рқ‘© , (sB , вүӨ-trans xвүӨA AвүӨB)
```



#### Closure properties of P

`P` is a closure operator.  This is proved by checking that `P` is *monotone*, *expansive*, and *idempotent*. The meaning of these terms will be clear from the definitions of the types that follow.


```agda
 H-expa : вҲҖ{в„“} вҶ’ {рқ’Ұ : Pred (Algebra Оұ ПҒбөғ)(a вҠ” ov в„“)} вҶ’ рқ’Ұ вҠҶ H в„“ рқ’Ұ
 H-expa {в„“} {рқ’Ұ}{рқ‘Ё} kA = рқ‘Ё , kA , IdHomImage

 S-expa : вҲҖ{в„“} вҶ’ {рқ’Ұ : Pred (Algebra Оұ ПҒбөғ)(a вҠ” ov в„“)} вҶ’ рқ’Ұ вҠҶ S в„“ рқ’Ұ
 S-expa {в„“}{рқ’Ұ}{рқ‘Ё} kA = рқ‘Ё , (kA , вүӨ-reflexive)

 P-mono :  вҲҖ{в„“ О№} вҶ’ {рқ’Ұ рқ’Ұ' : Pred (Algebra Оұ ПҒбөғ)(a вҠ” ov в„“)}
  вҶ’        рқ’Ұ вҠҶ рқ’Ұ' вҶ’ P{ОІ = Оұ}{ПҒбөғ} в„“ О№ рқ’Ұ вҠҶ P в„“ О№ рқ’Ұ'

 P-mono {в„“}{О№}{рқ’Ұ}{рқ’Ұ'} kk {рқ‘©} (I , рқ’ң , (kA , Bвү…вЁ…A)) = I , (рқ’ң , ((О» i вҶ’ kk (kA i)) , Bвү…вЁ…A))

 open _вү…_
 open IsHom

 P-expa : вҲҖ{в„“ О№} вҶ’ {рқ’Ұ : Pred (Algebra Оұ ПҒбөғ)(a вҠ” ov в„“)} вҶ’ рқ’Ұ вҠҶ P в„“ О№ рқ’Ұ
 P-expa {в„“}{О№}{рқ’Ұ}{рқ‘Ё} kA = вҠӨ , (О» x вҶ’ рқ‘Ё) , ((О» i вҶ’ kA) , Goal)
  where
  open Algebra рқ‘Ё using () renaming (Domain to A)
  open Algebra (вЁ… (О» _ вҶ’ рқ‘Ё)) using () renaming (Domain to вЁ…A)
  open Setoid A using ( refl )
  open Setoid вЁ…A using () renaming ( refl to reflвЁ… )

  toвЁ… : A вҹ¶ вЁ…A
  (toвЁ… вҹЁ$вҹ© x) = О» _ вҶ’ x
  cong toвЁ… xy = О» _ вҶ’ xy
  toвЁ…IsHom : IsHom рқ‘Ё (вЁ… (О» _ вҶ’ рқ‘Ё)) toвЁ…
  compatible toвЁ…IsHom =  reflвЁ…

  fromвЁ… : вЁ…A вҹ¶ A
  (fromвЁ… вҹЁ$вҹ© x) = x tt
  cong fromвЁ… xy = xy tt
  fromвЁ…IsHom : IsHom (вЁ… (О» _ вҶ’ рқ‘Ё)) рқ‘Ё fromвЁ…
  compatible fromвЁ…IsHom = refl

  Goal : рқ‘Ё вү… вЁ… (О» x вҶ’ рқ‘Ё)
  to Goal = toвЁ… , toвЁ…IsHom
  from Goal = fromвЁ… , fromвЁ…IsHom
  toвҲјfrom Goal = О» _ _ вҶ’ refl
  fromвҲјto Goal = О» _ вҶ’ refl


 V-expa : вҲҖ в„“ О№ вҶ’ {рқ’Ұ : Pred (Algebra Оұ ПҒбөғ)(a вҠ” ov в„“)} вҶ’ рқ’Ұ вҠҶ V в„“ О№ рқ’Ұ
 V-expa в„“ О№ {рқ’Ұ} {рқ‘Ё} x = H-expa {a вҠ” в„“ вҠ” О№} (S-expa {a вҠ” в„“ вҠ” О№} (P-expa {в„“}{О№} x) )
```


We sometimes want to go back and forth between our two representations of subalgebras of algebras in a class. The tools `subalgebraвҶ’S` and `SвҶ’subalgebra` are made for that purpose.


```agda
module _  {Оұ ПҒбөғ ОІ ПҒбөҮ в„“ О№ : Level}{рқ’Ұ : Pred (Algebra Оұ ПҒбөғ)(Оұ вҠ” ПҒбөғ вҠ” ov в„“)}
          {рқ‘Ё : Algebra Оұ ПҒбөғ}{рқ‘© : Algebra ОІ ПҒбөҮ} where

 S-вү… : рқ‘Ё вҲҲ S в„“ рқ’Ұ вҶ’ рқ‘Ё вү… рқ‘© вҶ’ рқ‘© вҲҲ S{Оұ вҠ” ОІ}{ПҒбөғ вҠ” ПҒбөҮ}(Оұ вҠ” ПҒбөғ вҠ” в„“) (Level-closure в„“ рқ’Ұ)
 S-вү… (рқ‘Ё' , kA' , AвүӨA') Aвү…B = lA' , (lklA' , BвүӨlA')
  where
  lA' : Algebra (Оұ вҠ” ОІ) (ПҒбөғ вҠ” ПҒбөҮ)
  lA' = Lift-Alg рқ‘Ё' ОІ ПҒбөҮ
  lklA' : lA' вҲҲ Level-closure в„“ рқ’Ұ
  lklA' = Lift-closed в„“ kA'
  subgoal : рқ‘Ё вүӨ lA'
  subgoal = вүӨ-trans-вү… AвүӨA' Lift-вү…
  BвүӨlA' : рқ‘© вүӨ lA'
  BвүӨlA' = вү…-trans-вүӨ (вү…-sym Aвү…B) subgoal

 V-вү… : рқ‘Ё вҲҲ V в„“ О№ рқ’Ұ вҶ’ рқ‘Ё вү… рқ‘© вҶ’ рқ‘© вҲҲ V{ОІ = Оұ}{ПҒбөғ} в„“ О№ рқ’Ұ
 V-вү… (рқ‘Ё' , spA' , AimgA') Aвү…B = рқ‘Ё' , spA' , HomImage-вү… AimgA' Aвү…B

module _  {Оұ ПҒбөғ в„“ : Level}
          (рқ’Ұ : Pred(Algebra Оұ ПҒбөғ) (Оұ вҠ” ПҒбөғ вҠ” ov в„“))
          (рқ‘Ё : Algebra (Оұ вҠ” ПҒбөғ вҠ” в„“) (Оұ вҠ” ПҒбөғ вҠ” в„“)) where
 private О№ = ov(Оұ вҠ” ПҒбөғ вҠ” в„“)

 V-вү…-lc : Lift-Alg рқ‘Ё О№ О№ вҲҲ V{ОІ = О№}{О№} в„“ О№ рқ’Ұ вҶ’ рқ‘Ё вҲҲ V{Оі = О№}{О№} в„“ О№ рқ’Ұ
 V-вү…-lc (рқ‘Ё' , spA' , lAimgA') = рқ‘Ё' , (spA' , AimgA')
  where
  AimgA' : рқ‘Ё IsHomImageOf рқ‘Ё'
  AimgA' = Lift-HomImage-lemma lAimgA'
```

 private a = Оұ вҠ” ПҒбөғ ; b = ОІ вҠ” ПҒбөҮ ; c = Оі вҠ” ПҒб¶ң

 LevelClosure-S : Pred (Algebra (Оұ вҠ” Оі) (ПҒбөғ вҠ” ПҒб¶ң)) (c вҠ” ov(a вҠ” b вҠ” в„“))
 LevelClosure-S = Level-closure{ОІ}{ПҒбөҮ} (a вҠ” в„“) (S в„“ рқ’Ұ)

 S-LevelClosure : Pred (Algebra (Оұ вҠ” Оі) (ПҒбөғ вҠ” ПҒб¶ң)) (ov(a вҠ” c вҠ” в„“))
 S-LevelClosure = S{Оұ вҠ” Оі}{ПҒбөғ вҠ” ПҒб¶ң}(a вҠ” в„“) (Level-closure в„“ рқ’Ұ)

 S-Lift-lemma : LevelClosure-S вҠҶ S-LevelClosure
 S-Lift-lemma {рқ‘Ә} (рқ‘© , (рқ‘Ё , (kA , BвүӨA)) , Bвү…C) =
