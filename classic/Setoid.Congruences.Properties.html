### Distributive and modular congruence lattices

This is the [Setoid.Congruences.Properties][] module of the [Agda Universal Algebra Library][].

[Setoid.Congruences.CompleteLattice][] assembled the congruence lattice of an
algebra.  This module names two properties that lattice may have ‚ÄĒ being
**distributive** and being **modular** ‚ÄĒ which the Maltsev conditions of congruence
distributivity (J√≥nsson) and congruence modularity (Day) characterize by the
existence of terms ([Setoid.Varieties.MaltsevConditions][]).

As in [Setoid.Congruences.CompleteLattice][], the lattice equations are stated at the
**absorbing** relation level `ūĚźč ‚Ąď‚āÄ = ūĚďě ‚äĒ ūĚď• ‚äĒ őĪ ‚äĒ ŌĀ ‚äĒ ‚Ąď‚āÄ`.  At this level the join
`_‚ą®_` (whose codomain otherwise bumps the level to `ūĚíą ‚Ąď`) lands back at the level of
the meet `_‚ąß_`, so both are operations on `Con ūĚĎ® (ūĚźč ‚Ąď‚āÄ)` and the equations type-check.

```agda
{-# OPTIONS --cubical-compatible --exact-split --safe #-}

open import Overture using ( ūĚďě ; ūĚď• ; Signature )

module Setoid.Congruences.Properties {ūĚĎÜ : Signature ūĚďě ūĚď•} where

-- Imports from Agda and the Agda Standard Library ----------------------------
open import Agda.Primitive  using () renaming ( Set to Type )
open import Level           using ( Level ; _‚äĒ_ ) renaming (suc to lsuc)

-- Imports from the Agda Universal Algebra Library ----------------------------
open import Setoid.Algebras.Basic          {ūĚĎÜ = ūĚĎÜ} using ( Algebra )
open import Setoid.Congruences.Basic       {ūĚĎÜ = ūĚĎÜ} using ( Con )
open import Setoid.Congruences.Lattice     {ūĚĎÜ = ūĚĎÜ} using ( _‚äÜ_ ; _‚ČĎ_ ; _‚ąß_ )
open import Setoid.Congruences.Generation  {ūĚĎÜ = ūĚĎÜ} using ( _‚ą®_ )
```

#### The absorbing relation level

```agda
-- The relation level at which both meet and join are operations on Con ūĚĎ®.
ŇĀ : Level ‚Üí Level ‚Üí Level ‚Üí Level
ŇĀ őĪ ŌĀ ‚Ąď‚āÄ = ūĚďě ‚äĒ ūĚď• ‚äĒ őĪ ‚äĒ ŌĀ ‚äĒ ‚Ąď‚āÄ
```

#### Congruence distributivity

An algebra `ūĚĎ®` is **congruence-distributive** (CD) when its congruence lattice
satisfies the distributive law `őł ‚ąß (ŌÜ ‚ą® Ōą) ‚ČĎ (őł ‚ąß ŌÜ) ‚ą® (őł ‚ąß Ōą)`.  (The reverse
containment half of this `‚ČĎ` is automatic in any lattice; the distributive law is the
forward containment `őł ‚ąß (ŌÜ ‚ą® Ōą) ‚äÜ (őł ‚ąß ŌÜ) ‚ą® (őł ‚ąß Ōą)`, but we state the full
symmetric `‚ČĎ` result for uniformity.)

```agda
module _ {őĪ ŌĀ : Level} (ūĚĎ® : Algebra őĪ ŌĀ)(‚Ąď‚āÄ : Level) where
  CongruenceDistributive : Type (lsuc (ŇĀ őĪ ŌĀ ‚Ąď‚āÄ))
  CongruenceDistributive = (őł ŌÜ Ōą : Con ūĚĎ® (ŇĀ őĪ ŌĀ ‚Ąď‚āÄ)) ‚Üí őł ‚ąß (ŌÜ ‚ą® Ōą) ‚ČĎ (őł ‚ąß ŌÜ) ‚ą® (őł ‚ąß Ōą)
```

#### Congruence modularity

An algebra `ūĚĎ®` is **congruence-modular** (CM) when its congruence lattice satisfies
the modular law: whenever `őł ‚äÜ Ōą`, `őł ‚ą® (ŌÜ ‚ąß Ōą) ‚ČĎ (őł ‚ą® ŌÜ) ‚ąß Ōą`.  Distributivity
implies modularity, so the congruence-distributive algebras form a subclass of the
congruence-modular ones.

```agda
  CongruenceModular : Type (lsuc (ŇĀ őĪ ŌĀ ‚Ąď‚āÄ))
  CongruenceModular = (őł ŌÜ Ōą : Con ūĚĎ® (ŇĀ őĪ ŌĀ ‚Ąď‚āÄ)) ‚Üí őł ‚äÜ Ōą ‚Üí őł ‚ą® (ŌÜ ‚ąß Ōą) ‚ČĎ (őł ‚ą® ŌÜ) ‚ąß Ōą
```
